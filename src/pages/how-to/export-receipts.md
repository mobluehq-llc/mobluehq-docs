Export gives you a portable record of receipts for audits or support tickets.

Planned surfaces:

- Copy session receipts as JSON from the receipt panel.
- Optional log directory on disk for long runs.

If your build does not show an export button yet, use the in-product **Report** flow so logs attach automatically.
