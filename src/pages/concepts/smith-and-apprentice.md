title: Smith and Apprentice
description: "The two names you see in the UI: the chat you type into, and the agent that edits code."
---

**Smith** is the chat surface. This is where you write intent, review answers, and steer the run.

**Apprentice** is the agent that plans work, calls tools, and applies edits. It is not a separate app: it is the name we use in docs and UI so you always know when the product is doing file changes versus simply answering a question.

When documentation says "ask Smith," that means type in the panel. When it says "Apprentice dispatched," that means the agent path is active and receipts will list tool use.
