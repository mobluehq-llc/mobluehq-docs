## Goal

Run one small change through Apprentice so you see how dispatch, tools, and receipts line up.

## Steps

1. Open a workspace folder you trust.
2. In Smith, describe a tiny task (for example, add a comment to one file).
3. Watch Apprentice pick tools. When it finishes, open the receipt list for that turn.
4. Confirm the files listed match what you expected.

## If something fails

See [Troubleshooting](/getting-started/troubleshooting) and the live [status page](https://mobluehq-status.pages.dev).
