## Provider errors

If Smith says a provider key is missing, open Settings and paste the key from your vendor console. Keys stay on your machine.

## Blank chat or stuck spinner

Check network and provider status. Visit the [public status page](https://mobluehq-status.pages.dev) for MOBLUEHQ-wide incidents.

## Desktop will not start

Confirm Rust and platform libraries match the Tauri README for your OS version. Re-run `npm install` after pulling updates.

## Still stuck

Use the in-product **Report** button so logs attach to your ticket. See [Report bugs](/how-to/report-bugs).
